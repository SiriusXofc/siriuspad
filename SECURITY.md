## Como reportar uma vulnerabilidade

Se você encontrou uma vulnerabilidade no SiriusPad, não abra uma issue pública.

As áreas mais sensíveis do projeto hoje são:

- execução de código via snippet runner
- armazenamento local do token do GitHub para Gist
- leitura e escrita de arquivos do usuário

Abra uma Security Advisory privada no GitHub:

https://github.com/Nic85796/siriuspad/security/advisories/new

Faço o possível para responder em até 72 horas.
