# Avisos Legais e Operacionais

O **SiriusPad** é um projeto público, open source e mantido por **SiriusX**.

## O que o app faz

- Salva notas localmente no seu computador.
- Pode executar comandos e snippets localmente quando você usa terminal ou runner.
- Pode falar com serviços externos apenas em fluxos opcionais, como exportar Gist ou checar atualização.

## O que isso significa na prática

- Revise comandos antes de executar.
- Não cole segredos em notas ou snippets sem entender o risco.
- Tokens do GitHub devem ser usados somente quando você realmente precisar da integração.
- Releases oficiais do projeto são distribuídas pelo GitHub do SiriusPad.

## Privacidade

- O app não depende de conta para uso normal.
- As notas continuam no seu disco.
- Integrações externas são acionadas por você.
